# `@yume-chan/android-bin`

Wrappers for Android built-in executables.

Currently it's bound to `@yume-chan/adb`, maybe one day it can be used with other executors.

## Implemented

* `bugreport`
* `bugreportz`
* `settings`
* Demo mode
* `logcat`
